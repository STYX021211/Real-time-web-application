# Real-time-web-application
The web app allows  users to communicate in real-time, exchange files, personalize their chat environment, and enjoy  gameplay all within a single platform. 
1.Real-time chat

2.Send and receive message

3.Multiline text

4.Email to hyperlink

5.Phone to WhatsApp link

6.Edit and delete sent message

7.Audio effects during message sent and received

8.Support more emoji (text to emoji, emoji box)

9.Text formatting (bold, italic, underline, monospace)

10.Timestamp (date and time, relative time)

11.Send and receive YouTube video

12.YouTube URL conversion

13.Send and receive image

14.Allow common image type

15.Local image sharing

16.Drag-and-drop image

17.Send multiple images

18.Send capture image from webcam

19.List all images

20.Fullscreen image

21.Send and receive file

22.Allow common file type (PDF, DOCX, etc)

23.Send multiple files

24.List all files

25.Changeable chatroom theme (color theme, background image)

26.Rude word filter (replace with ***)

27.Duplicated username prevention

28.Random username

29.Avatar options (icon, local image file, webcam)

30.User list (avatar and username)
